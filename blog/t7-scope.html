Scope


What 'scope' is, and how it works in JavaScript.

Scope refers to what variables (including objects and functions) are visible/accessible while a piece of code is running. Global scope can be visualized as a parent container to numerous local scopes. Every time a function is declared it creates a local scope within it. Variables defined within functions cannot be accessed in the global scope, however within local scopes any variable defined in global scope is visible and able to be referenced.
