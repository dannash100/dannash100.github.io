Can you think of an analogy to describe HTML, CSS and the DOM? How would you describe it to you non-technical friend?

  • HTML: The structure and content of a website
  • CSS: Uses specific syntax and rules to change what your website looks like
  • DOM: The Document Object Model is the branching structure of the code interpreted from HTML so your computer can work with it

What is meant by boxifying design?

  • Drawing a box around every element in a deisgn to devide it into its constituent parts. This organized approach makes it easier when writing HTML

What is the box model?

  • Every HTML element is a box and includes a series of additional boxes that wrap around it. The area around the content is called the padding, followed by a border and finally a margin, the attributes and sizes of these can be changed using CSS
