JavaScript Language


In the JavaScript language, describe all the use cases for each of:

parentheses ():

1. Execute a function. The parentheses can be empty or hold parameter names to be fed into the function. 2. Grouping and containing parameters i.e Separating blocks of code. 3. Controlling the order of evaluation in expressions i.e maths equations. 4. To push and pop: ways of removing or adding new elements in and out of an Array.

brackets []:

1. Surround the contents of an Array. 2. To access the properties of an array or object. Using [x] where x is a numerical value starting at 0 which refers to the first element of the array.

braces {}:

1. Define the content of an object bodies. 2. Surround conditional and loop bodies.

single quotes '' & double quotes ""::

1. When defining strings, used for storing and manipulating text 2. Mostly interchangeable except you can use both single quotes within double quotes (and vice versa) without the string escaping (ending preemptively).
