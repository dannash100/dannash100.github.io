Explain the difference between accessing data from arrays and objects.

To access data in an array use indexes. Written in brackets indexes specify the position of the entry in the array that the user wishes to access. Indexes use numerical values starting at 0 for the first entry in the array.

To access date in an object use the command objectName. followed by name of the property the user wishes to access, if it has a space in its name use [propertyName]. To access nested objects chain together dot and bracket notations to get to the desired property.

What a wireframe is and why we use it.

A wireframe is a simplistic sketch-like design of a page that includes mock-ups of various screen formats. This makes it easier to visualise as we implement our design into HTML.

Explain what functions are and why they are useful.

Functions are reusable groups of code that are executed when called upon (invoked). They receive values and return values. In this way they can be thought of as little machines that process incoming material and output a product.
